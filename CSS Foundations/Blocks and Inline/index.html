Blocks and Inline

Elements can be displayed in two default forms: as blocks or inline.

Depending on which display type, the element will have a different box model.

Blocks

Most elements that we've covered are block type.

  • Headers
  • Paragraphs

Blocks appear stacked on one another, from top to bottom

Blocks elements always appear on new lines and take up all the width of the page.

Divs

Divs are generic containers that are used to organize other elements

They have the block display type

Inline

Some elements are displayed inline.

    Anchors (links)

Inline elements continue from the previous, hence why these are called inline.

Padding and margin behave differently with inline elements.

Spans

Spans are generic containers for elements, but is an inline element itself.
