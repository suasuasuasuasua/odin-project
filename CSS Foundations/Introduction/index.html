    Specificity

    With CSS, there is an ordering, based on selectors, to which style is applied.

    1. Inline
    2. ID
    3. Class
    4. Type

    This is a sort of tiebreaker for conflicts, and 'rules' with more selectors will take precedence over ones with less.

    If there are equal number of inline, then ID is checked; if there are an equal number of ID, then class is checked; if there are more class selectors in one rule than another, then the majority wins.
