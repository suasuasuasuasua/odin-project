Axes

The orientation of items in a flex container can be controlled using the flex-direction property.

We should interpret two axes: the major and cross axis.

  • The major axis is the direction that the content is going: horizontal is left to right; vertical is up to down.

Horizontal

Horizontal is the default direction for flex items in a flexbox.

Vertical

To change the orientation to column-wise, we can say flex-direction: column
