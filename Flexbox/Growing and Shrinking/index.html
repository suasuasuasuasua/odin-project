Growing and Shrinking

The Flex Shorthand

"flex" is a property that you can give to elements.

flex: 0 is shorthand for:

Equivalently, we might say "flex 1 1 0"

  • flex-grow: 1
  • flex-shrink: 1
  • flex-basis: 0

flex: auto is shorthand for:

Equivalently, we might say flex 1 1 auto

  • flex-grow: 1
  • flex-shrink: 1
  • flex-basis: auto

Flex-Grow

Flex-Grow expects a single number as its value.

This is the growth factor.

flex: 1 means that each element is going to grow the same amount

flex: 2 would tell an element to grow twice the size of the other flex elements in the flexbox.

Flex-Shrink

Flex-Shrink is the same idea of Flex-Grow, but sets the shrink factor instead

Flex-Shrink only works if the side of the parent flex container is smaller than the width of the combined flex elements.

If we don't want an element to shrink, we say flex: 0.

Flex-Basis

Flex-Basis sets the initial size of a flex item

Any growing or shrinking starts from that base value

flex-basis: auto is a common way to fill it out if we want to respect defined width values
