Introduction to Flexbox

Flexbox is a relatively new technology to HTML that allows us to move elements around on a webpage.

When it was first introduced, it was revolutionary.

Flex Box

Flexbox is a way to arrange items into rows and columns.

The items will grow and shrink, or flex, based on some simple rules we can define.

The width of these divs depends on 1) how many there are, and 2) how big the viewport is.

Flex Containers

A flex container is any element that has "display: flex" on it

Likewise, a flex item is any element that is a child that lives in a flex container.

Interestingly, an element can be both a flex container and a flex item at the same time.
